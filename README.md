# Firefox

## Usage

```puppet

# from the stable channel
include firefox

# from the nightly channel
include firefox::nightly

```

## Required Puppet Modules

* `boxen`

## Development

Write code. Run `rake build` to test it. Check the `script`
directory for other useful tools.
